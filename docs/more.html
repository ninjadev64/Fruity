Fruity Points Index

The Fruity Points Index is a common database for Fruity and its partner bots to synchronise points.
It is accompanied by the Fruity Points API which allows anyone to query data from it.
Currently, the Points Index partner bots are Fruity and OwenTheBot.
You can apply for Points Index partner by contacting me on Discord.
